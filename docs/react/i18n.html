Internationalization

  • LocaleProvider
  • LocaleContext
  • i18n sample

The default language of antd@2.x is English as of yet. If you want to use other languages, you can follow the instructions below.

LocaleProvider#

choerodon-ui provides a React Component LocaleProvider for configuring choerodon-ui locale text globally.

import { LocaleProvider } from 'choerodon-ui';
import frFR from 'choerodon-ui/lib/locale-provider/fr_FR';

return (
  <LocaleProvider locale={frFR}>
    <App />
  </LocaleProvider>
);

LocaleContext#

choerodon-ui/pro provides LocaleContext for configuring choerodon-ui/pro locale text globally.

import { localeContext } from 'choerodon-ui/pro';
import en_US from 'choerodon-ui/pro/lib/locale-context/en_US';

localeContext.setLocale(en_US);

Note: fr_FR is the filename, follow below.
